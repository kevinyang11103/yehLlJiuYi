# SpingBoot电影会员管理系统 java430

## 项目概述

本项目是基于SpringBoot的电影会员管理系统，使用前后端分离的架构，后端采用SpringBoot与MyBatis，前端则基于Vue.js和Element-UI构建。

## 技术栈

- 后端：SpringBoot + MyBatis
- 数据库：MySQL
- 前端：Vue.js + Element-UI

## 功能模块

### 会员模块

- **会员列表**：显示系统中所有已注册会员的信息。
- **查询**：通过姓名、性别或ID号对会员进行查询。
- **登录**：提供会员登录功能，并显示会员卡列表。
- **注册**：会员注册功能。

### 会员卡模块

- 会员卡相关业务逻辑。

### 电影票模块

- 电影票相关业务处理。

### 会员关怀

- 会员关怀活动与策略。

### 数据统计

- 系统数据统计与展示。

## 系统角色

- 系统管理员
- 会员用户

## 运行环境

- 后端：Java 1.8+
- 前端：现代浏览器
- 数据库：MySQL 5.7+

## 部署步骤

1. 安装数据库MySQL，并创建相应的数据库。
2. 导入项目提供的SQL文件，进行数据初始化。
3. 部署后端服务，确保SpringBoot应用正常运行。
4. 构建前端项目，并部署至Web服务器。
